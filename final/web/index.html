Introduction
Music: Project! is a voice control system for mac OS X. We use some signal comparison techniques to detect the operation sounds from talking. We don't need to get the "words" we say, but the voice we produce. So, we can record some voices -- even not words, and use these to trigger some instructions we often use, e.g., play music, open a pdf file, ..., etc. For more details, please see the "Structure" tab or contact us. Thanks!

Techniques:
  • Detect real-time voice stream
  • Voice commands for simple instructions
  • User-defined voice for instructions
